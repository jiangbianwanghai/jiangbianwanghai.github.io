让你的网站支持谷歌和火狐浏览器的通知提醒

2018-06-04 /
  • 消息推送
  • 通知提醒
 / 点击 / 阅读耗时 2 分钟

消息推送在現在的網絡中已經成為主流。它可以解決人們實時獲取最新消息的方案。比如，客戶A在瀏覽一個網站，他關注的作者這個時候正好發佈了一篇文章，以前他必須刷新頁面，通過拉取的方式才能看到這位作者的文章。但是有了消息推送，只要作者發佈一篇文章，她就可以理解收到一篇 推送消息。這樣做，保證了時效性，也提高了用戶的閱讀體驗。

現在的主流瀏覽器，都提供了消息通知接口，開發人員可以通過 javascript 來調取這些接口，實現pc端的消息推送。
